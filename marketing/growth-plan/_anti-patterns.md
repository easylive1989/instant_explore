# 此階段不要做的事

| 不要做 | 為何看起來有吸引力 | 為何在 Early（前流量期）是錯的 |
|--------|--------------------|------------------------------|
| 投放付費廣告（Meta / Google Ads） | 快速觸達、立即有數字 | 無廣告預算工具；漏斗中段（落地頁→安裝→訂閱）尚未驗證，買來的流量會直接漏光 |
| 同時開 TikTok / YouTube Shorts | 「多平台曝光感覺穩」、素材現成 | 單人產能有限；IG 的 reach→profile 漏水還沒修好，複製到新平台等於複製同一個洞。先把 IG 做深 |
| 大規模投入 SEO 部落格 | 長期複利、每日故事素材現成 | 回報週期 3–6 個月起跳；此階段時間應花在天級回饋的管道（IG CTA、ASO）。Month 3 僅播種 2–3 篇 |
| 把每日故事做成付費電子報 | 「內容都有了，直接收費」 | 2026-07-10 已決策：受眾池為 0 時免費→付費數學不成立，且放棄 in-situ 差異化。電子報是免費漏斗，不是付費產品 |
| 用開發新功能取代行銷時間 | 寫 code 比行銷有掌控感 | 現在的瓶頸是「沒人看到產品」，不是「產品不夠好」。功能再多，3 個安裝也驗證不了任何假設 |
| SDR / cold email 開發 | B2B 教科書方法 | B2C App，用戶在 App Store 與 IG，不在信箱 |
| 過早啟動 A/B 測試 | 「數據驅動」 | 落地頁 ~1 訪客/天，任何 A/B 都是假訊號；等 web 新訪客有可測樣本量再說 |
| 追求 IG 粉絲數虛榮指標 | 成長儀表板好看 | 北極星是 reach→profile→link 的轉換與安裝，不是粉絲數；粉絲多但不點擊 = 0 |
