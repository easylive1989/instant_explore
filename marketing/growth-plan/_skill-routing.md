# Skill Routing — 下一步用哪些 skill

```
Week 1–2（P0：修 IG 漏水）
  lorescape-manual-daily-story / lorescape-daily-reel
      → 產製時內建結尾 CTA 卡與 caption 首行 CTA
  lorescape-reels-planner
      → 疑問句 hook 設為固定格式；8/3 期末檢核
  marketing-social
      → CTA 句式與 bio 文案產出
  marketing-gate
      → 所有對外文案（bio、CTA、ASO 摘要）品質檢查

Week 2–4（P1：自有管道 + 意圖流量）
  marketing-repurpose
      → 每日故事 → 電子報格式（含固定 App CTA 段落）
  marketing-landing-page
      → 定價/試用透明度、Wikipedia 異議區塊、5.0★ 徽章、
        新增「訂閱電子報」低門檻 CTA
  marketing-gate
      → ASO 標題/副標/描述前 3 行檢查

Month 2（分發 + 借力）
  marketing-repurpose
      → 每週 1–2 篇 PTT / Dcard / FB 社團原生故事文
  marketing-brand
      → KOL briefing 用的品牌訊息一致性
  marketing-weekly-audit / lorescape-metrics
      → 每週五四個北極星數字

Month 3（轉換 + 播種）
  marketing-seo-audit
      → blog 起手前的技術 SEO 檢查
  marketing-cro
      → 累計安裝有樣本後，重跑漏斗稽核
  marketing-monthly-audit
      → 對照本計畫月度覆盤
```

排除：任何廣告投放、email outreach、cold-outreach 工具。
