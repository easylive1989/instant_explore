## 寫作備註

- 第一行是「鉤子」：IG 在貼文展開前只會顯示開頭幾十字，所以第一句必須能讓人想點「展開」。
- 中段用「你有沒有過這種時刻 / 我們做的事很簡單」對話式結構，比官方介紹文好讀。
- 「田野誌」自己起一個欄目名稱，後續發文可以延續品牌語彙（例如「田野誌 #003｜京都八坂神社的火祭」）。
- Hashtag 分三層：品牌（#Lorescape）、中文中流量（#旅行 #自由行）、英文國際流量（#TravelApp #HiddenGems）。
- 表情符號用得克制：🎧 📱 🌏 三個就夠了，呼應「聲音 / App / 世界」。
